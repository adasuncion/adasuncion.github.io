Applied Computing Technology

The curriculum is a four-year degree program that focuses on the practical aspects of the science of computing. The program develops core skills in multimedia technology, web/mobile development and network/system administration. Furthermore, it prepares students with supervisory and management skills. The curriculum is derived from the BS Industrial Technology major in Computer Technology curriculum.

Register to our degree program.

Network Administrator

This provides competency pathway in installing, managing, and configuring computers, network, and server systems (Computer Systems Servicing NCII).

Web Developer

This provides competency pathways to utilize software methodologies, develop responsive web designs, create interactive websites and develop website backend systems. These competencies lead to job opportunities as front-end developer, back-end developer, or full-stack developer, and mobile developer

Multimedia designer

This provides competency pathways in visual graphic designing (VGD NCIII), photography (Photography NCII), video production and animation (Animation NCII).
